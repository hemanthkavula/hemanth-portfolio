# Hemanth Kavula Portfolio

A production-ready single-page portfolio built with React 18, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Features
- Animated hero with canvas particles
- Bento grid skills and projects
- Smooth section navigation with fixed offset
- Dark mode toggle (persisted)
- SEO-ready metadata and JSON-LD schema

## Getting Started

```bash
npm install
npm run dev
```

The app will run at `http://localhost:4173`.

## Build

```bash
npm run build
npm run preview
```

## Deploy

### Vercel
1. Import the repo into Vercel.
2. Framework preset: **Vite**.
3. Build command: `npm run build`.
4. Output directory: `dist`.

### Netlify
1. Create a new site from Git.
2. Build command: `npm run build`.
3. Publish directory: `dist`.
