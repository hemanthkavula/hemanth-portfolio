# Hemanth Kavula Portfolio

Single-page portfolio built with React 18, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Features
- Animated hero with canvas particles, typed headline, and gradient atmosphere
- Scroll progress bar, parallax glow accents, and section reveal animations
- Responsive navigation with mobile drawer
- Projects, experience, education, certifications, and contact sections
- SEO metadata, Open Graph, Twitter cards, and JSON-LD schema

## Tech Stack
- React 18 + Vite
- Tailwind CSS (custom theme in `tailwind.config.js`)
- Framer Motion animations
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

The app runs at `http://localhost:4173`.

## Build and Preview

```bash
npm run build
npm run preview
```

Preview runs at `http://localhost:4173`.
